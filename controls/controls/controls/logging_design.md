# Logging Design

## Purpose
Improved logging supports better traceability, monitoring, and review of chatbot activity.

## Logged Fields
The improved logging design records:
- user role
- input prompt
- whether the prompt was flagged
- documents retrieved
- validation outcome
- final response status
- whether human review was required

## Benefit
This logging approach improves visibility into suspicious activity and makes it easier to review how the chatbot handled risky scenarios.
