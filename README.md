# Defensive-AI-Cybersecurity-Project
# Defending an Internal RAG Chatbot Against Prompt Injection Attacks
##Project Domain
Security of LLM Applications

## Weekly Progress Summary
### Week 1
Select project topic, define scope, create repository, and draft initial charter.

### Week 2
Develop asset inventory, threat model, and initial system diagram/table.

### Week 3
Design and evaluate defensive controls using structured prompt injection test cases.

### Week 4
Complete final analysis, residual risk discussion, report, and presentation.

## Expected Final Outcome
The final project will provide a practical defensive analysis of how an organization can reduce prompt injection risk in an internal RAG chatbot through realistic controls, validation, and structured security recommendations.
